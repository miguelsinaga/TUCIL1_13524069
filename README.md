# Queens Solver with Region Constraints


## a. Requirement dan Instalasi

### Requirement:
- **Compiler**: GCC (g++) dengan support C++11 atau lebih tinggi
- **OS**: Windows, Linux, atau MacOS (yang mendukung g++)
- **Storage**: Minimal 5 MB untuk project files

### Instalasi:
- **Windows**: Instalasi MinGW-w64 atau TDM-GCC
- **Linux**: `sudo apt-get install g++` (Debian/Ubuntu)
- **MacOS**: `brew install gcc` atau Xcode Command Line Tools

Tidak ada library eksternal yang diperlukan - program menggunakan C++ standard library saja.

## b. Cara Mengkompilasi Program dan Menjalankan program

### Menggunakan Makefile:
```bash
make clean
make 
make run
```
